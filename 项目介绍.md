# 项目介绍：校园二手书交易平台（BookMart）

## 项目概述

本项目是一个面向高校学生的**微信二手书交易平台**，由前端小程序和后端服务两部分组成。

---

## 子项目一：Book_Market（后端服务）

### 是什么

基于 **Node.js** 开发的后端 API 服务，专为微信小程序提供数据支撑。

### 主要功能

- **微信登录**：对接微信小程序登录接口，完成用户身份验证
- **用户管理**：个人信息（昵称、学校、年级）的查看与修改
- **书籍系统**：书籍列表浏览、详情查看、搜索
- **交易流程**：购物车、收藏夹、订单管理
- **二手书提交**：用户拍照上传二手书 → 提交审核申请
- **书单匹配**：智能匹配用户所需书籍（`/api/book-plan`）
- **管理员后台**（内置 Web 页面 `/admin/index.html`）：
  - 超级管理员：账号管理、审核分配、书籍状态调整
  - 审核员（`admin1/admin2...`）：查看待审核书籍、评定质量等级与定价、批准或拒绝

### 技术栈

| 分类 | 技术 |
|------|------|
| 运行时 | Node.js |
| 认证 | JWT |
| 微信对接 | WECHAT_APPID / WECHAT_SECRET |
| 部署建议 | Nginx 反向代理 + HTTPS |

---

## 子项目二：Book_Market_Mini（微信小程序前端）

### 是什么

基于腾讯 **TDesign 零售行业模板**改造的微信小程序前端，提供完整的电商购物页面体验。

### 主要功能

- 首页展示、商品列表、商品详情
- 购物车与结算页面
- 个人中心（订单、收藏、个人信息）
- 售后流程页面
- 共 28 个完整页面

### 技术栈

| 分类 | 技术 |
|------|------|
| 框架 | 微信小程序原生 + TDesign 组件库 |
| 语言 | JavaScript + WXSS |
| 代码规范 | ESLint + Prettier |

---

## 项目关系

```
BookMart（BM）
├── Book_Market         ← 后端服务（Node.js API）
└── Book_Market_Mini    ← 微信小程序前端（TDesign）
```

后端部署于服务器后，小程序在 `app.js` 中配置好 API 域名即可对接。
